    Statements¶

    Function Definitions¶

    Sxpy version 

    (deco [decorator1
       decorator2]
  (def function-name [arg1 arg2 *args :kwarg1 value1 :kwarg2 value2 **kwargs]
    "docstring"
    body1
    body2
    (return value)))
    Python version 
    @decorator1
@decorator2
def function_name(arg1, arg2, *args, kwarg1=value1, kwarg2=value2, **kwargs):
    """docstring"""
    body1
    body2
    return value

    Async Function Definitions¶

    Use async-def instead of def.

    Class Definition¶

    Sxpy version 

    (deco [decorator1
       decorator2]
  (class class-name [base-class1 base-class2]
    "docstring"
    body))
    Python version 
    @decorator1
@decorator2
class class_name(base_class1, base_class2):
    """docstring"""

    body

    del¶

    Sxpy version 

    (del x y z ...)
    Python version 
    del x, y, z, ...

    Assignment¶

    Sxpy version 

    (= x value)
(= [x y *args] [value1 value2 value3 value4, ...]) ; destructuring assignment
    Python version 
    x = value
[x, y, *args] = [value1, value2, value3, value4, ...]
# then args = [value3, value4, ...]

    Augmented Assignment¶

    Sxpy version 

    (+= x value)
(-= x value)
(*= x value)
(/= x value)
...
    Python version 
    x += value
x -= value
x *= value
x /= value
...

    Type Annotations¶

    Simple Type Annotations¶

    Sxpy version 

    (= x ^type)
    Python version 
    x: type

    Assignment with Type Annotations¶

    Sxpy version 

    (= a ^int 1)
    Python version 
    a: int = 1

    Function Definition with Type Annotations¶

    Sxpy version 

    (def a [b ^int c] ^float
  (return (+ b c)))
    Python version 
    def a(b: int, c) -> float:
    return b + c

    Pydantic Example¶

    Sxpy version 

    (class User [BaseModel]
  (= id ^int)
  (= name ^str "John Doe")
  (= signup_ts ^(| datetime None))
  (= tastes ^(sub dict (, str PositiveInt))))
    Python version 
    class User(BaseModel):
    id: int
    name: str = "John Doe"
    signup_ts: datetime | None
    tastes: dict[str, PositiveInt]

    if¶

    Sxpy version 

    (if clause
    then
    else)
    Python version 
    if clause:
    then
else:
    else

    do block¶

    If you want to use multiple statements in then-body or else-body, you can use do block. Sxpy version 

    (if clause
    (do then1 then2)
    (do else1 else2))
    Python version 
    if clause:
    then1
    then2
else:
    else1
    else2

    with¶

    Simple with¶

    Sxpy version 

    (with [expression]
  body)
    Python version 
    with expression:
    body

    with with alias¶

    Sxpy version 

    (with [expression as target]
  body)
    Python version 
    with expression as target:
    body

    Asynchronous with¶

    Use async-with instead of with.

    match¶

    Sxpy version 

    (match x
  (case "Relevant"
        ...)
  (case None
        ...)
  (case [1 2]
        ...)
  (case [1 2 *rest]
        ...)
  (case [*_]
    ...)
  (case {1 _ 2 _}
    ...)
  (case {**rest}
    ...)
  (case (Point2D 0 0)
    ...)
  (case (Point3D :x 0 :y 0 :z 0)
    ...)
  (case [x] as y
    ...)
  (case _
    ...)
  (case (| [x] y)
    ...))
    Python version 
    match x:
    case "Relevant":
        ...
    case None:
        ...
    case [1, 2]:
        ...
    case [1, 2, *rest]:
        ...
    case [*_]:
        ...
    case {1: _, 2: _}:
        ...
    case {**rest}:
        ...
    case Point2D(0, 0):
        ...
    case Point3D(x=0, y=0, z=0):
        ...
    case [x] as y:
        ...
    case _:
        ...
    case [x] | y:
        ...

    try¶

    Sxpy version 

    (try
  body1
  body2
  (except [Exception1]
    body3
    body4)
  (except [Exception2 as e]
    body5
    body6)
  (else
    body7
    body8)
  (finally
    body9
    body10))
    Python version 
    try:
    body1
    body2
except Exception1:
    body3
    body4
except Exception2 as e:
    body5
    body6
else:
    body7
    body8
finally:
    body9
    body10

    except* clause¶

    Sxpy version 

    (try
  (raise BlockingIOError)
  (except* [BlockingIOError as e]
    (print (repr e))))
    Python version 
    try:
    raise BlockingIOError
except* BlockingIOError as e:
    print(repr(e))

    from module import name1
from module import name2 as alias
from module import name3
