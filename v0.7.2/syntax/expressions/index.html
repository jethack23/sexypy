    Expressions¶

    Operators¶

    Boolean Operators¶

    Sxpy version 

    (and expr1 expr2 ...)
    Python version 
    expr1 and expr2 and ...

    Binary Operators¶

    Sxpy version 

    (+ n1 n2 ...)
    Python version 
    n1 + n2 + ...

    Unary Operators¶

    Sxpy version 

    (- n)
    Python version 
    -n

    Comparison Operators¶

    ==¶

    Sxpy version 

    (== expr1 expr2 expr3 ...)
    Python version 
    expr1 == expr2 == expr3 == ...

    >¶

    Python version 

    (> expr1 expr2 expr3 ...)
    Sxpy version 
    expr1 > expr2 > expr3 > ...

    :=¶

    Sxpy version 

    (:= name value)
    Python version 
    (name := value)

    labmda¶

    Sxpy version 

    (lambda [arg1 arg2 ...] body)
# or syntactic sugar fn
(fn [arg1 arg2 ...] body)
    Python version 
    lambda arg1, arg2, ...: body

    if expression¶

    Sxpy version 

    (ife test then other)
    Python version 
    then if test else other

    Collection Literals¶

    List¶

    Sxpy version 

    [item1 item2 ...]
    Python version 
    [item1, item2, ...]

    Dictionary¶

    Sxpy version 

    {key1 value1 key2 value2 ...}
    Python version 
    {key1: value1, key2: value2, ...}

    Set¶

    Sxpy version 

    {, item1 item2 ...}
    Python version 
    {item1, item2, ...}

    Tuple¶

    Sxpy version 

    (, item1 item2 ...)
    Python version 
    (item1, item2, ...)

    Comprehensions¶

    List Comprehension¶

    Sxpy version 

    [(* 2 x) for x in (range 10) if (== (% x 2) 0)]
    Python version 
    [2 * x for x in range(10) if x % 2 == 0]

    Dictionary Comprehension¶

    Sxpy version 

    {(* 2 x) (* 3 x) for x in (range 10) if (== (% x 2) 0)}
    Python version 
    {2 * x: 3 * x for x in range(10) if x % 2 == 0}

    Set Comprehension¶

    Sxpy version 

    {, (* 2 x) for x in (range 10) if (== (% x 2) 0)}
    Python version 
    {2 * x for x in range(10) if x % 2 == 0}

    Generator Expression¶

    Sxpy version 

    ((* 2 x) for x in (range 10) if (== (% x 2) 0))
    Python version 
    (2 * x for x in range(10) if x % 2 == 0)

    await¶

    Sxpy version 

    (await expr)
    Python version 
    await expr

    yield¶

    Sxpy version 

    (yield expr)
    Python version 
    yield expr

    yield from¶

    Sxpy version 

    (yield-from expr)
    Python version 
    yield from expr

    Function Call¶

    Sxpy version 

    (func-name arg1 arg2 *args :kwarg1 value1 :kwarg2 value2 **kwargs)
    Python version 
    func_name(arg1, arg2, *args, kwarg1=value1, kwarg2=value2, **kwargs)

    Formatted String Literals¶

    Sxpy version 

    (= name "World")
f"Hello, {(* 2 name)}!"
    Python version 
    name = "World"
f"Hello, {2 * name}!"

    Attribute¶

    Sxpy version 

    (. obj attr)
    Python version 
    obj.attr

    Method Call¶

    Sxpy version 

    (.method-name obj arg1 arg2 *args :kwarg1 value1 :kwarg2 value2 **kwargs)
    Python version 
    obj.method_name(arg1, arg2, *args, kwarg1=value1, kwarg2=value2, **kwargs)

    Subscription¶

    Simple Subscription¶

    Sxpy version 

    (sub obj i1 i2)
    Python version 
    obj[i1][i2]

    Tuple Subscription¶

    Sxpy version 

    (sub obj (, i1 i2 ...))
    Python version 
    obj[i1, i2, ...]

    Slice¶

    Slice Subscription¶

    Sxpy version 

    (sub obj [: start stop step])
    Python version 
    obj[start:stop:step]

    Slice with Emptiness¶

    Sxpy version 

    (sub obj [: start])
(sub obj [: _ stop])
(sub obj [: start _ step])
    Python version 
    obj[start:]
obj[:stop]
obj[start::step]
